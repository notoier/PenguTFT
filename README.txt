# PenguTFT 🐧
Bot de Discord para mostrar y sincronizar el rango de Teamfight Tactics.

## 🚀 Funcionalidades
- `/link Nombre#TAG` — Vincula tu cuenta de Riot.
- `/tft` — Muestra tu rango actual de TFT.
- Asigna roles automáticos según el rango.
- Crea roles TFT con colores personalizados si no existen.
- Actualiza y limpia roles automáticamente cada 6 horas.
